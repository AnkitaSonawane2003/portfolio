personal portfolio website built using modern web technologies to showcase skills, projects, achievements, and contact information. This project highlights both front-end development and integration with back-end services.

🚀 Features

About Section – Introduction ,academic and professional background.

Skills Section – Technical skills 

Projects Section – Detailed showcase of personal and academic projects.

Certificates Section – Collection of certifications and achievements.

Contact Section – Contact form with email functionality (Nodemailer).


🛠️ Tech Stack

Frontend:

React.js

HTML, CSS, JavaScript 

Backend 

Node.js, Express.js

Nodemailer (for email service)

Deployment:

Vercel / Render
